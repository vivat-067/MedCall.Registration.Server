# МедВызов: регистрация заявок вызова СМП

Демо‑версия прототипа медицинской информационной системы.  
Реализация проекта MedCall.Registration.Server предоставляет API для доступа клиента и включает функциональность генерации тестовых данных.

## Описание проекта

«МедВызов» — демонстрационная версия прототипа системы для регистрации заявок на вызов скорой медицинской помощи (СМП). Проект предназначен для:
* демонстрации архитектуры МИС;
* тестирования клиентских приложений через API;
* прототипирования решений в сфере медицинской информатизации.

## Функциональные возможности

* генерация тестовых данных для отладки и демонстрации функциональности с указанием основных данных (адрес, причина вызова, состояние пациента и т. д.);
* предоставление RESTful API для тестирования интеграции с клиентскими приложениями;

## Технологии и стек

* **Платформа:** .NET 10;
* **Серверная часть:** ASP.NET;
* **Язык программирования:** C#;
* **Формат данных:** JSON;
* **Интерфейс взаимодействия:** REST API.

## Запуск и настройка

### Предварительные требования

* Установленный [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0);
* Среда разработки (Visual Studio или VS Code с расширением C#).

### Локальный запуск
Приложение сервера доступно по адресу: http://localhost:5285.

## Использование API

После запуска сервера API доступно по базовому URL: `http://localhost:5285/api`.

Для интерактивной документации и тестирования API используется **Swagger UI**. После запуска приложения откройте в браузере:

`http://localhost:5285/swagger`

### Доступные эндпоинты

**Управление заявками:**

* `GET /api/MedicalAssistanceCalls` — получить список всех заявок;
* `GET /api/MedicalAssistanceCalls/{id}` — получить заявку по ID;
* `POST /api/MedicalAssistanceCalls` — создать новую заявку (отправьте JSON с данными заявки);
* `PUT /api/MedicalAssistanceCalls/{id}` — обновить заявку;
* `DELETE /api/MedicalAssistanceCalls/{id}` — удалить заявку.
